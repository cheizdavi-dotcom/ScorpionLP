# **App Name**: ScorpionLP Landing Page

## Core Features:

- Hero Section Design: Implement a visually striking hero section with a welcome badge, impactful headline, supportive subheadline, and a prominent call-to-action button to capture visitor attention immediately.
- Headline Animation: Introduce subtle animations to the headline to draw the eye and emphasize key phrases like 'ESTRUTURA QUE VENDE' using a vibrant red color.
- Call-to-Action Button: Design a prominent, pill-shaped call-to-action button with a unique red glow to encourage user interaction and drive conversions effectively.

## Style Guidelines:

- Primary color: A vibrant red (#E53935) to highlight key elements such as 'ESTRUTURA QUE VENDE' and the CTA button, symbolizing urgency and action.
- Background color: Light gray (#F5F5F5) to provide a clean, neutral backdrop that enhances readability and visual hierarchy.
- Accent color: Dark gray (#424242) for the subheadline and body text, ensuring a sophisticated contrast against the light background.
- Headline font: 'Playfair', a modern serif with an elegant, high-end feel, suitable for headlines; Body text: 'PT Sans' a humanist sans-serif.
- Use a small font size (around 12px) with increased letter-spacing (2px) for the 'WELCOME TO SCORPIONLP' badge to create a luxurious, sophisticated feel.
- Center-align the welcome badge at the top for a balanced and formal introduction, emphasizing ScorpionLP's elite status.
- Implement a subtle glowing animation effect on the main CTA button to attract attention and encourage clicks.
